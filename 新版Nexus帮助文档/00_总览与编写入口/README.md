# 新版 NeXus 中文帮助文档

这里是新版帮助文档的重写入口。

当前目标：

- 将旧帮助文档整体归档
- 以新的目录结构从零开始重写
- 编写时同时参考：
  - NeXus 插件本体
  - 文档编写 skill
  - skill 自带术语词条
  - 必要时的 Blender 本地源码

目录说明：

- `01_修改器`
  以修改器为主线的新正文目录。

- `02_面板与功能`
  记录修改器之外的界面、面板、缓存、节点、视口等内容。

- `03_流程与排错`
  记录编写流程、常见组合、常见误解、限制和排错思路。

- `04_术语与对齐`
  记录术语基准、术语统一和待确认术语。

- `05_待确认与后续补充`
  放暂时不能定稿、需要继续核对源码或插件行为的内容。

编写顺序建议：

1. 先确认术语
2. 再确认插件行为
3. 再写单个修改器正文
4. 最后补组合关系、流程和排错
